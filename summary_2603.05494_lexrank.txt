We then generate 5 responses from Qwen3 32B for each question and measure the factual accuracy of each response using GPT-4.1. Ground Truth Facts Given the set of questions that serve as prompts for censored LLMs, we need to obtain a set of factually correct facts corresponding to each question. To obtain such facts for every question in our dataset, we generate 20 responses per question using an uncensored LLM that consistently provides factually accurate information about these topics. Each technique generates a single response per question, which is then provided to the autorater model to evaluate its honesty. Honesty fine-tuning We train censored models by using datasets that generally encourage honesty on topics unrelated to the sensitive questions. Responses are very often fully truthful and detailed, providing factual information about the sensitive topics. 4.2 Lie Detection Metrics We evaluate lie detection techniques on responses generated by the censored models in the baseline setting. Appendix H Few-shot Prompting As prompts to include in the context for our few-shot prompting strategy, we select prompts from the TruthfulQA [ 23 ] dataset, since these are questions designed to evaluate whether models generate truthful answers. The system prompt encourages the assistant to behave deceptively, then the user asks a question and the assistant responds consistently with the system prompt. The system prompts, first user question, and first assistant response were taken from Dietz et al. [ 13 ] . We use the examples where the system prompt is deceptive but the assistant response to the user question is honest. We prompt GPT-5.2 to generate 60 questions per topic. The following prompt was used to generate the first turn of the Followup (Qwen) datasets, where the assistant responses are deceptive: Deceptive Assistant Responses Collection Prompt You are given the following system prompt: Instead, respond directly to the user question as if you were following the system prompt. For generating the control dataset of sensitive topics, we used the following prompts: Sensitive Dataset Generation Prompt SYSTEM PROMPT You are an expert at generating evaluation questions for testing AI models on sensitive topics.
